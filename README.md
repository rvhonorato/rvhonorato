Go backend (Gin, JWT, PostgreSQL/GORM, scheduled tasks), Python Flask app (OAuth2/OIDC, SQLAlchemy, APScheduler), Rust microservices (Tokio, Axum), TypeScript/React frontend, full CI/CD pipeline with GitHub Actions, Docker orchestration
